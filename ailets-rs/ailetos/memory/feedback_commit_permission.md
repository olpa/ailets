---
name: Always ask before committing
description: Never run git commit without explicit user permission
type: feedback
---

Do not run `git commit` unless the user explicitly asks to commit (e.g. "commit", "mark and commit", "save it").

**Why:** User rejected an unsolicited commit and said "don't commit without permission."

**How to apply:** After finishing a change, present the result and wait. Only commit when the user explicitly requests it.
